Sasto Open Source Contribution

Contribute to Sasto

Sasto is an open-source online marketplace where anyone can sell products without any restrictions. We welcome contributions from the community to make this platform even better!

How to Contribute

  1. Fork the Sasto repository on GitHub.
  2. Clone the forked repository to your local machine.
  3. Create a new branch for your feature or bug fix: git checkout -b feature-branch-name
  4. Make your changes and commit them.
  5. Push the changes to your GitHub repository: git push origin feature-branch-name
  6. Create a Pull Request on the original Sasto repository.

Need Help?

If you have any questions or need help contributing, feel free to reach out to us on support@sasto.com.
